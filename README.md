# The Moovie App

## Objectifs fonctionnels  
Dans ce projet, nous allons créer l'application netFlix2 (The new Netflix). L'objectif est d'exploiter la base de données TheMoovieDB (https://developers.themoviedb.org/3) afin de permettre aux utilisateurs de l'application de visualiser, noter et voir la bande annonce des films de la base de données.
